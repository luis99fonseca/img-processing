◆ write_bin()

void write_bin ( ImageBin *  image,
char *  file_name 
)

Method to save a Binary image in a PPM file.

Parameters
imageA pointer to the Binary image.
file_nameImage PPM filename.

Here we play with the bits as following: each ImageBin has a BinaryPixel, in which the color is at least significative bit (aka, of order 0), thus, in each loop, we shift that bit to the left (most significative bit posible), in order to form a byte legible for this format; Once we form a byte, it is written, and a new one is started to be formed;
